Happy-Sad Image Recognition with CIFAR-10 Exploration

Objective: Develop a neural network-based model to classify images into two emotions: happy and sad. Additionally, explore and preprocess the CIFAR-10 dataset.

Sections:

  1. Setting up Dependencies: Initial setup to load necessary libraries and tools required for the project.
  2. Exploring the Happy-Sad Dataset: A detailed exploration and visualization of the image dataset, understanding its structure, and providing insights about the data.
  3. Scaling the Data: Pre-processing step to ensure the image data is normalized or standardized.
  4. Splitting the Data: Dividing the dataset into training, evaluation, and test sets.
  5. Building the Neural Network Model: Designing and implementing the neural network architecture.
  6. Training: Initiating the model training process.
  7. Plotting Performance: Visualizing the model's training performance.
  8. Evaluate: Assessing the trained model's performance.
  9. Testing the Model: A final test of the model's classification ability.
  10. CIFAR-10 Data Preprocessing: Loading the CIFAR-10 dataset using Keras, exploring its structure, normalizing images, and one-hot encoding labels.

Github Link: https://github.com/huygg1004/Image_Classification.git
