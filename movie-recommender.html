MOVIE RECOMMENDATION SYSTEM

  • User-based collaborative: Using the Pearson correlation function for computing similarities between users
  • Item-based: Using the cosine similarity for computing similarities between items
  • Output: Select a user from the dataset, and for this user, show the 20 most relevant movies that the recommender suggests
  • Group recommendation: Aggregation approach - the average method vs. the least misery method
  • Output: Produce a group of 3 users, and for this group, show the top-20 recommendations
  • Methods for producing explanations for:
    • Granularity case of both atomic and group cases
    • Position absenteeism case
  • Technologies Used: Numpy, pandas, matplotlib, data preprocessing, seaborn, sklearn
  • Dataset: MovieLens 100K https://grouplens.org/datasets/movielens/
