SOCIAL NETWORK ANALYSIS

Description

This project presents a comprehensive social network analysis project. It begins by creating basic undirected and directed graphs to visualize relationship structures. Network measures such as degrees, diameter, edge density, reciprocity, centrality, and betweenness are computed, offering insights into the network's characteristics. Real data is imported from a CSV file, and a network is constructed, with node labels and degrees calculated. A histogram illustrates the distribution of node degrees, and various network diagrams are used to visualize the network's structure. Finally, hub and authority scores are calculated to assess node importance, and community detection is performed to identify clusters within the network, making your analysis both informative and visually appealing.

Force-Directed Layout (Fruchterman-Reingold)

The first diagram uses a force-directed layout algorithm known as Fruchterman-Reingold. It positions the vertices in a way that minimizes overlapping edges. Nodes with different degrees are colored using a rainbow palette, and their sizes vary based on their degrees, with higher-degree nodes being larger.

Optimized Graph Layout (Graphopt)

The second diagram employs an optimized graph layout (Graphopt) that arranges vertices to optimize the overall layout based on certain criteria. Similar to the previous diagram, it uses a rainbow color palette for nodes and adjusts node sizes according to their degrees.

Kamada-Kawai Layout

In the third diagram, we utilize the Kamada-Kawai layout algorithm. This layout algorithm aims to visually position vertices in an efficient manner, enhancing the clarity of the network structure. As with the others, node colors and sizes provide insights into the network's characteristics.
